# Algebraic Identites
1. $(a + b)² = a² + 2ab + b²$
1. $(a-b)² = a² - 2ab + b²$
1. $a²–b² = (a + b)(a – b)$
1. $a² + b² = (a - b)² + 2ab$
1. $(a + b + c)² = a²+b²+c²+2ab+2bc+2ca$
1. $(a + b)³ = a³ + 3a²b + 3ab² + b³$
1. $(a – b)³ = a³ - 3a²b + 3ab² - b³$
1. $a³−b³ = (a−b)(a²+b²+ab)$
1. $a³+b³ = (a+b)(a²+b²-ab)$
